Основные возможности

Роутинг

Buffalo использует библиотеку Gorilla для работы с роутами, сессиями и файлами cookie... Вероятно, существуют и более быстрые роутеры, но этот определенно самый мощный!

Шаблонизатор

Создавайте ваши шаблоны при помощи Plush, используя синтаксис схожий с Rails. Расширяйте базовые возможности шаблонизатора при помощи собственных расширений. А если он вам не нравится, то всегда можно использовать стандартный html/template или даже написать свой!

Инструменты Buffalo

Writing a web application module always begins with the same tasks. Buffalo provides you a simple toolbox, the buffalo command, to generate many parts of your app and run usual tasks.

Extend the toolbox with plugins, using the language you want!

Тестирование

Тестирование может быть скучной задачей, но Buffalo помогает вам в создании тестовых сценариев. Action, resource, model generators create test templates for the code you just generated.

You can run them all using a simple command from the toolbox.

"Горячая" перезагрузка

Code, save, refresh. Use the buffalo dev command to rebuild your app, from backend to frontend, and just see the changes live!

Frontend pipeline Optional

Use the Webpack-generated configuration to build your frontend assets, so you can optimize both backend and frontend.

Модели / ORM Опционально

Тесная интеграция с pop provides a simple way to handle database and common related tasks.

Поддерживаемые базы данных: MySQL/ MariaDB, PostgreSQL, CockroachDB, SQLite.

Фоновые воркеры / задачи Опционально

If you're familiar with Rake tasks from Ruby, you'll be right at home using Grift. Seeding a database, running a cleaning job are now at hand!

Running background tasks are easy too, using the Background Workers.
